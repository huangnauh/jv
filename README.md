# jv
jv (for jsonviewer) helps you view your JSON.

[![asciicast](https://asciinema.org/a/123606.png)](https://asciinema.org/a/123606)

## Installation
```
go get -u github.com/maxzender/jv
```

## Example usage
```
cat file.json | ./jv aaa
```
